*** Git

**** Clone

#+BEGIN_SRC sh
cd <some dir>
git clone --recurse-submodules https://github.com/PaulRaffer/Blackjack.git
cd Blackjack
#+END_SRC

**** Run

Run in default browser ($BROWSER):
#+BEGIN_SRC sh
. run.sh
#+END_SRC
or to run in a specific browser:
#+BEGIN_SRC sh
. run.sh <browser>
#+END_SRC
